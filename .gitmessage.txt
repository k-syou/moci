# type list
# feat : 새로운 기능 추가
# fix : 버그 수정
# docs : 문서 수정
# test : 테스트 코드 추가
# refact : 코드 로직, 기능 수정(리팩토링)
# style : 코드 의미에 영향을 주지 않는 변경사항(세미콜론, 오탈자, 띄어쓰기 등)
